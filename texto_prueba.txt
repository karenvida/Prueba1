Algunos comandos importantes de github
	- git clone <url> - copia repositorios
	- git add - envia archivos al staging area donde los archivos estan listos para ser subidos a la nube
	- git add* incluye TODOS los cambios
	- git commit -m "message" crea la captura de los archivos en staging area. El mensaje debe ser descriptivo de la captura
	- git push - envia las capturas que tengamos hacia la nube para sincronizarlo a la nube.
	- git status - permite ver en que estado se encuentra nuestro repositorio
	- git pull - cuando algun compañero haga algún cambio y yo quiero actualizar mi repositorio local debo jalar lo actualizado con git pull
